# Workbook 5.3 — Cross-platform interpreted program

## Topic

Binary reuse and source code reuse.

## Task

Develop a program using an interpreted programming language that can run on any platform where the corresponding interpreter is installed.

## Description

This project is written in Python.

The program reads a text file named `input.txt` and prints simple statistics:

- number of lines;
- number of words;
- number of characters;
- the longest word.

The program is cross-platform because it does not use platform-specific APIs. It works on Windows, Linux and macOS if Python is installed.

## Run

```bash
python main.py
```
